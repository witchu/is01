# ตัวละครในนิยาย: **Midnight Proof**
> นิยายโรแมนติก 52 ตอน | ฉากหลัง: กรุงเทพฯ ยุคปัจจุบัน

---

## ตัวเอก

### ลิน — ไพลิน จันทร์เรือง (Pailin "Lin" Chanrueang)
| รายละเอียด | ข้อมูล |
|---|---|
| **อายุ** | 25 ปี |
| **อาชีพ** | บาร์เทนเดอร์ หัวหน้าชิฟท์กลางคืน ที่บาร์ "Noir" — cocktail bar สไตล์ speakeasy ย่านทองหล่อ |
| **รูปร่างหน้าตา** | ผมสั้นข้างเดียว ผมยาวอีกข้าง (undercut asymmetric) สีผมธรรมชาติ ดวงตาคม มีรอยปานเล็กๆ ใต้ตาซ้าย ร่างกายแข็งแรงจากการยืนทำงานทุกคืน มือคล่องแคล่ว |
| **ภูมิหลัง** | เกิดในครอบครัวชนชั้นกลาง จังหวัดเชียงใหม่ ย้ายมากรุงเทพฯ ตอนอายุ 19 เพื่อเรียนออกแบบกราฟิก แต่ลาออกกลางคัน เพราะครอบครัวมีปัญหาการเงิน เริ่มทำงานบาร์ตั้งแต่ 21 ปี จนกลายเป็นมืออาชีพ |
| **นิสัยใจคอ** | สังเกตการณ์เก่ง อ่านคนได้แม่น พูดน้อยแต่คิดมาก มีอารมณ์ขันแบบ dry humor ปกป้องตัวเองด้วยการทำเย็นชา แต่ข้างในอ่อนไหวง่าย ภักดีกับคนที่รัก อย่างยิ่ง |
| **ของที่ชอบ** | เพลง jazz ยุค 60s, การทดลองผสมค็อกเทลสูตรใหม่, หนังสือจิตวิทยา, แมวส้ม (เลี้ยงแมวชื่อ "มาตุลี"), กาแฟดำตอนเช้า |
| **ของที่เกลียด** | คนโกหก, เสียงดังในตอนเช้า, การถูกมองว่าเป็นแค่ "บาร์เทนเดอร์สวย", ความคลุมเครือในความสัมพันธ์ |
| **ความลับ** | เคยมีความรักครั้งแรกที่บอบช้ำมาก ทำให้กลัวการผูกพัน มีสมุดบันทึกสูตรค็อกเทลที่ซ่อนความรู้สึกตัวเองไว้ในชื่อค็อกเทล |
| **ความฝัน** | เปิดบาร์เล็กๆ เป็นของตัวเอง สักวัน |

### คนที่หนึ่ง — กร (กรวิชญ์ สิทธิวรรณ / Korawich "Korn" Sittiwat)

| รายละเอียด | ข้อมูล |
|---|---|
| **อายุ** | 28 ปี |
| **อาชีพ** | สถาปนิกและผู้ร่วมก่อตั้งสตูดิโอออกแบบ "Form & Soul" |
| **รูปร่างหน้าตา** | สูง หน้าตาเรียบร้อย ผมสั้นเนี้ยบ มักสวมเสื้อขาวหรือโทนนูด นิ้วมือยาวและแข็งแรง (วาดแบบ) สายตาลึกและสงบ |
| **ภูมิหลัง** | ครอบครัวฐานะดี พ่อเป็นนักธุรกิจ แม่เป็นศิลปิน ถูกคาดหวังมาตลอด เรียนเก่ง มีวินัย เคยหมั้นกับผู้หญิง/ผู้ชายคนหนึ่งที่พ่อแม่จัดให้ แต่เลิกกันก่อนแต่งงาน กำลังค้นหาว่าตัวเองต้องการอะไรจริงๆ |
| **นิสัยใจคอ** | สงบ ตรงไปตรงมา ไม่พูดมาก แต่ถ้าพูดจะตรงประเด็นเสมอ เป็นคนมีหลักการสูง รักสวยรักงาม ใส่ใจรายละเอียด โรแมนติกแบบ subtle — ไม่ได้โชว์แต่แสดงออกด้วยการกระทำ |
| **จุดเด่น** | ความมั่นคงและความน่าเชื่อถือ ทำให้รู้สึกปลอดภัย |
| **ของที่ชอบ** | สถาปัตยกรรม brutalist, วิสกี้ single malt, หนังสือปรัชญา, เสียงฝน, ต้นไม้ในกระถาง |
| **ของที่เกลียด** | ความวุ่นวายที่ไม่จำเป็น, การโกหกเพื่อรักษาน้ำใจ, ความล่าช้า |
| **ค็อกเทลที่ลินทำให้** | Old Fashioned (คลาสสิก นิ่ง ลึก) |
| **มาบาร์เพราะ** | ออฟฟิศใกล้เคียง มาคนเดียวบ้าง กับลูกค้าบ้าง จนกลายเป็น regular |

### คนที่สอง — ภู (ภูริ ตะวัน / Phuri "Phu" Tawan)

| รายละเอียด | ข้อมูล |
|---|---|
| **อายุ** | 27 ปี |
| **อาชีพ** | ช่างภาพ travel & lifestyle อิสระ เขียนบทความให้นิตยสารและบล็อกส่วนตัวที่มีผู้ติดตามหลักแสน |
| **รูปร่างหน้าตา** | ผมหยักศกธรรมชาติยาวพอประมาณ ผิวแทนจากการเดินทาง มักสวมเสื้อลินินหลวมๆ กับกางเกงผ้า มีกล้องฟิล์มแขวนคอเสมอ รอยยิ้มติดตัว |
| **ภูมิหลัง** | เติบโตในครอบครัวที่ขยับขยายบ้านบ่อย พ่อเป็นนักทูต ทำให้ชินกับการย้ายที่อยู่และการพบเจอผู้คนใหม่ เรียนจบด้านการสื่อสาร แต่เลือกใช้ชีวิตนอกกรอบ ไม่ยึดติดที่ใด |
| **นิสัยใจคอ** | เปิดเผย สนุกสนาน พูดเก่ง เข้าหาคนได้ง่าย ดูเหมือนไม่จริงจัง แต่จริงๆ แล้วเข้าใจความรู้สึกคนอื่นลึกมาก มีอารมณ์อ่อนไหวที่ซ่อนอยู่หลังรอยยิ้ม กลัวการตั้งรกราก |
| **จุดเด่น** | ทำให้ชีวิตสดใสและน่าตื่นเต้น มองโลกในแง่ดี |
| **ของที่ชอบ** | แสงทองยามเช้า, ตลาดสด, ดนตรีใต้ดิน, สตรีทฟู้ด, ฟิล์มขาวดำ |
| **ของที่เกลียด** | ความจำเจ, ตารางเวลาที่แน่นเกินไป, การถูกกักขัง (ทั้งกายและใจ) |
| **ค็อกเทลที่ลินทำให้** | Aperol Spritz (สดใส มีชีวิตชีวา แต่มีความขมแอบซ่อน) |
| **มาบาร์เพราะ** | กลับมากรุงเทพฯ ระหว่างเดินทาง และชอบบาร์นี้ตั้งแต่ครั้งแรกที่เห็น |

### คนที่สาม — ธีร์ (ธีร์ สัตยาพงศ์ / Thee Sattayaphong)

| รายละเอียด | ข้อมูล |
|---|---|
| **อายุ** | 30 ปี |
| **อาชีพ** | อดีตนักดนตรีวงดัง ปัจจุบันเป็น music producer / composer ให้ภาพยนตร์ |
| **รูปร่างหน้าตา** | ผมยาวประบ่า มักมัดหลวมๆ รอยสักที่แขน สีหน้าแห้งเกินอายุ แต่ดวงตาคมและมีชีวิต มักสวมเสื้อดำ |
| **ภูมิหลัง** | โด่งดังมากช่วงอายุ 22–26 แต่เพื่อนร่วมวงเสียชีวิตจากอุบัติเหตุ วงแตก ธีร์รู้สึกผิดและถอยห่างจากวงการ ใช้ชีวิตเงียบๆ แต่ยังคงสร้างงานดนตรีอยู่เบื้องหลัง |
| **นิสัยใจคอ** | เงียบขรึม สังเกตการณ์สูง ไม่เชื่อใจง่าย แต่เมื่อไว้วางใจแล้วจะซื่อสัตย์อย่างสุดโต่ง มีความโกรธที่ยังคุกรุ่น และความอ่อนโยนที่ยังหลงเหลือ พูดน้อย แต่ทุกคำมีความหมาย |
| **จุดเด่น** | ความลึกและความเข้มข้นของอารมณ์ รู้สึกได้ว่าเขาเข้าใจในสิ่งที่ไม่พูด |
| **ของที่ชอบ** | เสียงกีตาร์เบา ดึกๆ, กาแฟเย็น, ฝนตก, ซีรีย์สารคดี, ลินคือคนแรกที่เขาคุยด้วยได้นานๆ |
| **ของที่เกลียด** | การถูกจดจำในฐานะ "อดีต", คนที่แสร้งทำเป็นเข้าใจ, เสียงปรบมือที่ดังเกินไป |
| **ค็อกเทลที่ลินทำให้** | Negroni (ขม แต่ซับซ้อน มีมิติ) |
| **มาบาร์เพราะ** | หนีจากอพาร์ตเมนต์ใกล้ๆ มาที่นี่เพราะเงียบกว่าที่อื่น จนกลายเป็นลูกค้าประจำ |

