# ตอนที่ 43 — Return

ลินกลับมาทำงานวันจันทร์

เปิดประตู เปิดไฟ เริ่มเช็คของ เตรียมบาร์ ทำงานที่คุ้นเคยมาสี่ปี

ทุกอย่างในบาร์เหมือนเดิม ชั้นขวด เก้าอี้ที่นั่ง แสงไฟ เสียงระบบปรับอากาศ กลิ่นไม้เก่าๆ ที่ผสมกับแอลกอฮอล์และน้ำหอมของลูกค้าหลายร้อยคน

แต่ลินรู้สึกต่างออกไปเล็กน้อย ไม่ได้เบากว่า ไม่ได้หนักกว่า แค่อยู่ในที่ของตัวเองมากกว่า

โมเห็นและพูดว่า "แกดูดีขึ้นนะ"

"ขอบคุณค่ะ"

"ไม่ได้ชม — แค่สังเกต"

ลินยิ้ม

กรมาตอนสี่ทุ่ม

ไม่ได้นัด แค่มา เหมือนที่กรมามาตลอด นั่งที่บาร์ที่นั่งเดิม มองลิน

ลินเดินมา "สวัสดีค่ะ"

"สวัสดีครับ" กรพูด เสียงปกติ ไม่มีความกังวลที่ต้องแสดงออก แค่สวัสดี

"เหมือนเดิมไหมคะ?"

กรพักสักครู่ แล้วพูดสิ่งที่ลินไม่ได้คาดไว้ "Old Fashioned ครับ แต่ถ้าคุณอยากแนะนำอะไรใหม่ก็ได้"

ลินหยุด มองกร

กรนั่งรออย่างสงบ ไม่ได้หมายความว่าสละสิทธิ์ แต่หมายความว่าเปิดพื้นที่ให้ลินตัดสินใจ

ลินพิจารณาสักครู่ แล้วหันไปที่ชั้นขวด มือเริ่มเลือก

Bourbon เดิม แต่ใส่น้ำตาลมะพร้าวแทนน้ำตาลขาว เพิ่มความอ่อนหวานที่ลึกกว่า บีบส้มลงในแก้วก่อนเท ให้น้ำมันเปลือกกระจาย

ไม่ใช่ Old Fashioned แบบเดิมทุกอย่าง แต่ยังเป็น Old Fashioned

วางลงตรงหน้ากร

กรหยิบขึ้น จิบ นิ่ง แล้วพยักหน้า

"อร่อยกว่าครับ"

ลินไม่ได้พูดว่าอะไร แค่ยิ้มเล็กน้อย แล้วเดินไปรับออร์เดอร์ที่โต๊ะอื่น

---

คืนนั้นผ่านไปเหมือนคืนปกติ ลินทำงาน ลูกค้าเข้าออก เพลงดัง เพลงเบา

แต่มีอะไรบางอย่างในตัวลินที่ตั้งอยู่กับตัวเองมากกว่าเดิม ไม่ได้สั่นไหวกับทุกสิ่งที่ผ่านเข้ามา ไม่ได้รู้สึกว่าต้องจัดการทุกอย่างในเวลาเดียวกัน

แค่ทำงานที่ตัวเองเลือกมาทำ ในบาร์ที่ตัวเองรักมาตลอด

และนั่นก็เพียงพอสำหรับคืนนี้
